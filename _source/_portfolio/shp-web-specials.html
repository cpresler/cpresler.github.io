Web Specials Project

I collaborated with a designer to build a page that allows users to more easily peruse Swanson's ongoing sales on desktop, tablet, and mobile.

Visit Page

I used JSP to dynamically pull in the top four items in each sale for easy viewing by the customer.

To make the page easier to navigate I created a custom sub-navigation that allows users to jump to the sale of their choice.
