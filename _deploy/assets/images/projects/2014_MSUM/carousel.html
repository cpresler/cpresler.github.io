jQuery, the web's most used JavaScript Library

What is a JavaScript Library?

A JavaScript library is a library of pre-written JavaScript which allows for easier development of JavaScript-based applications, especially for AJAX and other web-centric technologies.

-- Wikipedia

Why would I use jQuery? (or any other JavaScript Library)

Pro's

  • Quicker development
  • Easier cross-browser compatability
  • Easier DOM Element selection
  • Greater structure helpful for beginners

Con's

  • Increases page load time
  • Adds to site "bloat"
  • Predefined tools can be a hinderence

So should I use jQuery for my project?

I decide for each project, but especially when you are just starting out it can be very helpful

Implementing jQuery:

Externally Hosted

Advantage: Easy implementation. Set it and forget it.

<script src="http://code.jquery.com/jquery-1.11.0.min.js"></script>

Place that code snippet above the closing body tag of your page. (It needs to be before any code that uses jQuery or that code will break.)

Internally Hosted

Advantage: Can be incorporated into your master script file to take advantage of caching.

Download files from: jquery.com

Make things happen once the page has loaded

In "vanilla" JavaScript:

window.onload = function() { /* stuff you want to happen here */ };

To do this in jQuery:

$('document').ready(function() { /*stuff you want to happen here*/ });

jQuery Methods

.addClass()

.addClass() does exactly what it sounds like. It adds the specified class to the selected element.

$('.slide').addClass('active');

You can add multiple classes at the same time, and classes containing variables.

More info

jQuery Methods

.removeClass()

.removeClass() is the exact opposite of .addClass(). It removes the specified class from the selected element.

$('.slide').removeClass('active');

With these two methods we can selectively add and take away classes containing any kind of css styles we want. We'll be using the 'active' css class to keep track of what slide we are on for our carousel.

More info

jQuery Methods

.css()

.css()allows us to apply any css styles directly to an element. The syntax for this one is a bit more complex because it can be used in several different ways:

  • To retrieve the value of a style, or styles: 
    $('.slide').css('background-color');
  • To apply and set styles: 
    $('.slide').css({'background-color': '#f1f1f1', 'border-radius': '5px'});

jQuery Methods

.next()

.next() retrieves the next matching sibling of the element selected. It also accepts a selector, and will find the next sibling that matches that selector. Very useful for down jumping through elements in the DOM

$('.slide').next();
More info

jQuery Methods

.prev()

.prev() is the opposite of next, in that it moves up the DOM. It retrieves the sibling before the element in the DOM.

$('.slide').prev();
More info

jQuery Methods

.animate()

.animate() creates transitions between css style changes. It allows you to set the styles you are changing to, the time duration of the change(in milliseconds), and even specify a function to happen at the end of the animation

$('.slide').animate({opacity: 0.25}, 5000, function() { 
      console.log('done animating') 
});
More info

jQuery Methods

.length

.length is a method that doesn't accept any parameters. It simply measures the length or number of a collection. You can take the length of strings to find the number of letters, the length of an array to find the number of items, etc.

$('.slide').length;
More info

jQuery Methods

.each()

.each() creates a loop that loops through each item selected. Generally you see this written with a function to be applied to each item.

$('.slide').each(function(index){ console.log('slide' + index); });
More info

jQuery Methods

.attr()

.attr() allows you to capture, set, or apply attributes and values to DOM elements.

$('.slide').attr('id', 'soap');

.append()

.append() takes the code you put between the () and adds it inside of the DOM element selected. Adding each additional element after the one before. (.prepend(), adds in the opposite order)
